# V2.5 Source Gap Enrichment Note

## Purpose

This note converts the V2.4 downstream concentration comparison into a concrete source-gap enrichment plan.

It does not add new external values. It identifies what evidence would be needed to strengthen the comparison without expanding beyond a bounded descriptive concentration analysis.

## Input

- data/processed/v24_downstream_concentration_comparison.csv

## Output

- data/curated/v25_source_gap_enrichment_plan.csv

## Boundary

This is a source-gap planning artifact. It is not a forecast, investment recommendation, policy recommendation, geopolitical ranking, bottleneck proof, vulnerability ranking, or complete supply-chain risk model.

## Priority source gaps

### High priority

- lithium: high_value_comparability_gap. Would test whether the observed large processing/refining concentration gap remains robust under clearer country-share evidence.
- manganese: definition_comparability_gap. Would determine whether the current proxy can become a comparable processing/refining metric or must remain definition-limited.

### Medium priority

- natural_graphite: medium_value_comparability_gap. Would improve confidence in the observed processing/refining concentration gap and clarify product-definition limits.

### Medium-low priority

- cobalt: precision_and_validation_gap. Would improve precision for minerals where mining and processing/refining concentration values are close.
- nickel: precision_and_validation_gap. Would improve precision for minerals where mining and processing/refining concentration values are close.

## Evidence acceptance criteria

A candidate source should only be used if it identifies:

- year;
- geography;
- product definition;
- denominator;
- country or producer share basis;
- source table, chart, dataset, or explicit source statement.

If those criteria are not met, the source may still be useful as context, but it should not be converted into a comparable concentration value.

## Claim boundary after enrichment

Even after source enrichment, the project should remain a bounded descriptive concentration comparison.

Still blocked:

- investment signal;
- policy recommendation;
- geopolitical ranking;
- bottleneck proof;
- disruption probability;
- company-level control;
- vulnerability ranking;
- complete supply-chain risk conclusion.

## Suggested next research sequence

1. Resolve manganese definition comparability first.
2. Strengthen lithium processing/refining country-share evidence because V2.4 shows a large gap.
3. Clarify natural graphite product-definition alignment.
4. Add precision sources for nickel and cobalt if comparable public tables are available.

## Interpretation

V2.5 does not change the V2.4 findings. It creates a targeted source-enrichment roadmap for turning the current public-source snapshot into a stronger, more traceable concentration-comparison study.
