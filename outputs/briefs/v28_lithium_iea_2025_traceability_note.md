# V2.8 Lithium IEA 2025 Traceability Note

## Purpose

This note resolves the highest-priority V2.7 lithium traceability action for the IEA 2025 primary report candidate.

V2.8 does not create a new lithium analytical model. It traces the existing V2.4 lithium processing/refining top-three value to an explicit IEA lithium table.

## Inputs

- data/processed/v24_downstream_concentration_comparison.csv
- data/curated/v27_lithium_evidence_extraction_register.csv

## Output

- data/curated/v28_lithium_iea_2025_traceability_register.csv

## Traceability result

The existing V2.4 lithium processing/refining value is source-traceable.

IEA Lithium 2025 reports the share of top three refining countries as 96% for 2024.

This matches the V2.4 lithium processing/refining top-three value.

IEA Lithium 2025 also reports a 2024 share of top three mining countries of 77%. This is useful as a cross-check, but it does not automatically replace the project's existing mining baseline because that would require a separate methodology decision.

## Accepted now

- V2.4 lithium processing/refining value can be marked as source-traceable to IEA Lithium 2025.
- The IEA 2025 lithium refining top-three value can be used to support the existing bounded descriptive concentration comparison.
- Historical and projection values can be stored as context only.

## Not accepted now

- no investment signal;
- no policy recommendation;
- no geopolitical ranking;
- no bottleneck proof;
- no vulnerability ranking;
- no disruption probability;
- no replacement of the mining baseline without a separate methodology decision;
- no trend claim from 2021/2024/2030/2040 without a separate annual comparison architecture.

## Claim boundary

V2.8 supports traceability for a bounded descriptive concentration comparison. It does not turn concentration into a validated risk model, forecast, investment conclusion, policy conclusion, or geopolitical ranking.

## Next work option

The next productive step is either to update a small internal traceability index or to repeat the same traceability process for manganese, where the evidence class is weaker and the definition comparability problem is more important.
