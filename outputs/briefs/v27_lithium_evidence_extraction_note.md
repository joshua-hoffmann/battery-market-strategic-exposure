# V2.7 Lithium Evidence Extraction Note

## Purpose

This note converts the V2.6 lithium source candidate plan into an extraction-readiness register.

V2.7 does not add a new lithium concentration value. It records whether each candidate source is ready for metric extraction, requires manual testing, or should remain context/baseline only.

## Input

- data/curated/v26_lithium_processing_refining_source_candidates.csv

## Output

- data/curated/v27_lithium_evidence_extraction_register.csv

## Extraction result

No new lithium refining or processing concentration value is accepted in this step.

The IEA Global Critical Minerals Outlook 2025 remains the primary report candidate, but the specific lithium 2024 top-three refining metric should only be accepted after locating the exact source table, chart, downloadable data, or explicit source statement.

The IEA Critical Minerals Data Explorer remains a data-tool candidate. It should be tested manually for exportable lithium processing/refining data with year, geography, stage, denominator, and share basis.

The IEA Critical Minerals topic page remains context only.

The IEA Global Critical Minerals Outlook 2024 remains a historical comparison candidate, not a current metric source.

USGS Lithium remains mining-baseline support only and should not be used as lithium refining evidence.
