# V2.14 Midstream Consolidation Note

## Purpose

This note consolidates the V2.4 to V2.13 processing/refining concentration and traceability line.

It does not add new mineral values, does not replace prior values, and does not create a public-facing claim.

## Current stage architecture

Per V2.12, the current project stage architecture is:

- upstream: V2.1 mine-production concentration;
- midstream: V2.4 to V2.13 processing/refining concentration and traceability;
- downstream: open future layer, not yet claimed.

## Consolidated midstream reading

Lithium is the strongest current midstream case because the existing V2.4 processing/refining top-three value is source-traceable after V2.8.

Manganese remains analytically useful but proxy-limited after V2.9 because the downstream value concerns battery-grade or high-purity manganese sulphate rather than a fully comparable general manganese processing/refining denominator.

Natural graphite remains a public-snapshot case after V2.13. The product-definition gap is clearer, but the existing V2.4 value 99 is not numerically accepted as source-verified. Natural graphite mining, natural flake graphite, spherical graphite, coated spherical graphite, synthetic graphite, battery-grade graphite, and anode material should not be treated as interchangeable denominators.

Cobalt and nickel remain lower-priority public-snapshot cases because their observed mining-versus-processing/refining gaps are small and therefore sensitive to rounding, denominator, and source-traceability precision.

## Accepted now

- the V2.4 to V2.13 line can be treated as the current midstream evidence line;
- lithium is source-traceable for the existing processing/refining top-three value;
- manganese is retained as proxy-limited;
- natural graphite is retained as product-definition-limited public snapshot;
- cobalt and nickel are retained as low-priority precision-gap public snapshots.
