# V2.12 Stage Architecture Decision Note

## Purpose

This note records the practical stage architecture decision after reviewing the legacy V2.1 to V2.3 line against the newer V2.4 to V2.11 traceability-controlled work.

It does not revise prior outputs, does not add new mineral values, and does not create a public-facing claim.

## Decision

V2.1 should be retained as the primary upstream block because it focuses on observed mine-production concentration and has the strongest visual explanation potential.

V2.4 to V2.11 should be treated as the current midstream mainline because the strongest current evidence concerns processing/refining concentration, source traceability, and product-definition comparability.

V2.2 should be retained as a legacy bridge from upstream-only concentration toward processing/refining comparison, but it is partly superseded by V2.4 to V2.11.

V2.3 should not be treated as the primary downstream module. It is evidence-limited directional context, especially for cobalt and nickel, and should be superseded, integrated, or kept as a legacy appendix rather than polished as a main stage.

True downstream is not yet built. A true downstream layer would require reliable raw data for battery or cell manufacturing, cathode/anode material production, EV battery demand, regional demand exposure, or end-market dependency.

## Reader-value implication

The strongest current story is:

- upstream: V2.1 mine-production concentration;
- midstream: V2.4 to V2.11 processing/refining concentration and traceability;
- downstream: open future layer, not yet claimed.
